# ÉPIKA - Sistema Integrado de Gestión (v0.1.0-alpha)

Sistema administrativo premium a medida diseñado para la gestión de inventario, ventas físicas y virtuales, abonos, garantías, co-inversiones con socios y análisis multimoneda para **ÉPIKA**.

## 🚀 Tecnologías Planificadas
* **Base de Datos:** PostgreSQL
* **Backend:** Python (FastAPI o Django por definir)
* **Frontend:** Por definir (enfocado en experiencia móvil ágil)

## 📁 Estructura del Proyecto
* `/docs`: Documentación técnica, requisitos de negocio y diagramas.
* `/app`: Código fuente del backend y frontend del sistema.

## 🤝 Roles del Sistema
* **Administrador Principal:** Control total de inventario, finanzas, socios y reportes.
* **Colaboradores autorizados:** Gestión diaria de ventas, abonos y registro de garantías.
